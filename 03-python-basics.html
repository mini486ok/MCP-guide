Python 기초 문법

MCP 서버를 만들 때 실제로 필요한 만큼만 빠르게 훑습니다. 이 장을 끝내면 이후 장의 모든 코드를 읽을 수 있습니다.

예상 학습 시간 · 20분 실습 권장
이 장의 목표

Python의 모든 문법을 가르치는 것이 아닙니다. MCP 서버 코드를 읽고 이해하는 데 필요한 최소한만 압축해서 소개합니다. 더 깊이 공부하고 싶다면 페이지 맨 아래 "더 배우기" 링크를 참고하세요.

1. 화면에 값 출력하기

제일 간단한 프로그램입니다.

python
print("안녕하세요!")
print(1 + 2)  # 3

# 뒤의 글자는 주석(사람이 읽는 메모)입니다.

2. 변수와 기본 타입

값에 이름을 붙여 저장하는 것이 변수입니다. Python은 값을 보고 타입을 자동으로 판단합니다.

python
station_name = "강남역"       # str  (문자열)
platform = 3                  # int  (정수)
distance_km = 1.8             # float(실수)
is_transfer = True            # bool (참/거짓: True 또는 False)

문자열은 f-string으로 다른 값을 끼워넣을 수 있습니다(매우 자주 씁니다).

python
msg = f"{station_name}의 승강장은 {platform}번입니다."
print(msg)  # 강남역의 승강장은 3번입니다.

3. 리스트와 딕셔너리

여러 값을 묶어 저장할 때 가장 많이 쓰이는 두 자료구조입니다.

리스트(list)

순서가 있는 값의 묶음입니다. [ ]를 씁니다.

python
stations = ["서울역", "시청", "종각", "종로3가"]
print(stations[0])      # "서울역" — 첫 번째는 0번
print(len(stations))    # 4
stations.append("종로5가")

딕셔너리(dict)

"키 → 값" 짝을 저장합니다. { }를 씁니다. 한 줄로 여러 속성을 표현할 때 씁니다.

python
station = {
    "name": "서울역",
    "line": 1,
    "lat": 37.5547,
    "lng": 126.9707,
}
print(station["name"])   # "서울역"
station["line"] = 4      # 값 수정
왜 중요한가?

MCP 서버가 AI에게 돌려주는 데이터는 대부분 딕셔너리(또는 딕셔너리의 리스트) 형태입니다. 이 두 가지만 자연스럽게 다룰 수 있으면 우리 가이드를 끝까지 따라올 수 있습니다.

4. 조건문과 반복문

들여쓰기(띄어쓰기 4칸)가 매우 중요합니다. 같은 들여쓰기가 "같은 묶음"을 뜻합니다.

python
speed = 85

if speed > 100:
    print("과속입니다")
elif speed > 80:
    print("주의 구간")
else:
    print("정상")

for name in stations:
    print(name)

5. 함수 정의

함수는 MCP Tool의 기본 단위입니다. 조금만 더 자세히 보겠습니다.

python
def add(a, b):
    return a + b

result = add(3, 5)
print(result)  # 8
  • def로 시작해서 함수 이름을 정합니다(여기선 add).
  • 괄호 안은 매개변수(이 함수가 받는 입력).
  • return으로 값을 돌려줍니다.

6. 타입 힌트 핵심

Python은 원래 타입을 굳이 쓰지 않아도 되지만, MCP에서는 타입을 반드시 적어줘야 합니다. 왜냐하면 FastMCP가 이 타입 정보를 읽어 "AI에게 알려줄 입력·출력 형식"을 자동으로 만들기 때문입니다.

python
def add(a: int, b: int) -> int:
    return a + b
  • a: int — "a는 정수"라고 알려주는 표시.
  • -> int — "이 함수는 정수를 반환"이라는 표시.

자주 쓰는 타입은 이 정도입니다.

타입의미예시 값
int정수3, -7
float실수1.8, 0.0
str문자열"강남역"
bool참/거짓True, False
list[str]문자열의 리스트["A", "B"]
dict[str, int]키는 문자열, 값은 정수인 딕셔너리{"x": 1, "y": 2}

7. Docstring 핵심

함수 바로 아래에 """ ... """(삼중 따옴표)로 적는 설명문입니다. FastMCP가 이 설명을 그대로 읽어 AI에게 전달하므로 MCP에서는 특히 중요합니다.

python
def add(a: int, b: int) -> int:
    """두 정수를 더해 그 합을 돌려줍니다.

    Args:
        a: 첫 번째 정수.
        b: 두 번째 정수.

    Returns:
        a + b 의 결과값.
    """
    return a + b
외우는 팁

MCP에서는 docstring을 "AI가 읽는 사용 설명서"라고 생각하세요. 잘 쓰면 AI가 내 함수를 더 똑똑하게 활용합니다. 이 부분은 05장에서 구체적으로 이어집니다.

8. import 와 실행 진입점

다른 파일이나 라이브러리의 기능을 가져다 쓰려면 import를 씁니다.

python
from fastmcp import FastMCP   # fastmcp 라이브러리에서 FastMCP 클래스만 가져오기
import math                   # math 모듈 전체 가져오기
print(math.pi)                # 3.141592...

파이썬 파일 맨 아래에 자주 보이는 마법의 줄:

python
if __name__ == "__main__":
    mcp.run()

"이 파일을 직접 실행했을 때만 mcp.run()을 돌려라"라는 뜻입니다. 다른 파일에서 import로 불러올 때는 건너뛰게 합니다. MCP 서버 파일에서는 거의 매번 이 패턴을 쓰게 됩니다.

9. 작게 연습해보기

실습 — 5분

다음 함수를 VS Code에서 practice.py로 만들고 직접 실행해 보세요. 입력을 바꿔가며 결과를 확인하세요.

python · practice.py
def greet(name: str, loud: bool = False) -> str:
    """이름을 받아 인사말을 돌려줍니다.

    Args:
        name: 인사할 대상의 이름.
        loud: True이면 대문자로 돌려줍니다.

    Returns:
        완성된 인사말 문자열.
    """
    message = f"안녕하세요, {name}님!"
    if loud:
        message = message.upper()
    return message


if __name__ == "__main__":
    print(greet("미나"))
    print(greet("Claude", loud=True))
